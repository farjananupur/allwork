নিকলী হাওর

নিকলী হাওর সত্যিই আশ্চর্যজনক যে প্রকৃতিতে কিছু সময়ের জন্য হারিয়ে যেতে চায়। আপনি যদি দ্বি-জলে ভ্রমণ করতে আগ্রহী হন তবে নিকলী হাওরের দিকে যান। নিকলী হাওর কিশোরগঞ্জ জেলার অন্তর্গত নিকলী উপজেলায় অবস্থিত। কিশোরগঞ্জ সদর থেকে নিকলী উপজেলার মধ্যে এর দূরত্ব প্রায় ২৫ কিলোমিটার। সাগরের স্বচ্ছ জলে ভাসমান দ্বীপের মতো ছোট ছোট শহর এবং মাছ ধরার প্রতি জেলেদের আবেশ, ছোট জলরাশি, রাতারগুলের মতো তাজা এবং সুস্বাদু হাওর স্যামন।

নিকলী হাওর দেখার উপযুক্ত সময়

আপনি যদি হাওরের অভ্যন্তরে জলের আকার দেখতে চান তবে বৃষ্টির মরসুমের পরে যাওয়া ভাল। সুতরাং, নিকলীতে যাওয়ার আদর্শ সময় হল জুলাই থেকে সেপ্টেম্বরের মধ্যে। সেপ্টেম্বরে অবশ্য পানির স্তর কম থাকে।

কিভাবে যাবেন নিকলী হাওরে

ঢাকার পাশাপাশি কিশোরগঞ্জ জেলা থেকে একদিনেই নিকলী হাওর পরিদর্শন করা যায়। ঢাকায়, আপনি ট্রেন বা বাসে নিকলী হাওর পৌঁছাতে পারেন।

কোথায় খাবে?

নিকলি যে এলাকায় অবতরণ করছে সেখানে ঠিক সেখানে কয়েকটি শীর্ষস্থানীয় খাদ্য প্রতিষ্ঠান রয়েছে। হোটেল ব্রিজ এবং ক্যাফে ওয়েভ সবচেয়ে উল্লেখযোগ্য। হাওর থেকে 100-200 টাকায় তাজা মাছ কেনা যায়।

Web Designer

Web designers plan, create and code internet sites and web pages, many of which combine text with sounds, pictures, graphics and video clips. A web designer is responsible for creating the design and layout of a website or web pages. It and can mean working on a brand new website or updating an already existing site.

Web Developer

A web developer is a programmer who develops World Wide Web applications using a client–server model. The applications typically use HTML, CSS and JavaScript in the client, PHP, ASP.NET (C#), Python, Node. js, Go or Java in the server, and http for communications between client and server.

Programmer

Programmers write code for computer programs and mobile applications. They also are involved in maintaining, debugging and troubleshooting systems and software to ensure that everything is running smoothly.

Grapic designer

A graphic designer is a professional within the graphic design and graphic arts industry who assembles together images, typography, or motion graphics to create a piece of design. A graphic designer creates the graphics primarily for published, printed, or electronic media, such as brochures and advertising.
